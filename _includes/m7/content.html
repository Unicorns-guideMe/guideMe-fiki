First, the user has to sign in its account. This allows the user to access its saved resources from any computer.

When a link is saved, the user can add one or more tags and a description, rate the quality of the resource and select its difficulty.

Links are saved in directories, based on their tags. After selecting a folder, the user can see the links saved, with their webpage titles. "See note" button opens the link in a new tab.

When the user searches a topic, the extension automatically searches in our database to find the best resources, which can be filtered by difficulty.

Future work

At the moment, the recommendation system doesn't bring much value to our product because we don't have enough data. As the number of users increases, we will be able to train a more complex algorithm and enhance this functionality, as well as add new ones like offering learning roadmaps.

Stay tuned!
